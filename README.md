## Project Purpose

This system enables Trade Ocean Namibia clients to:
- Register import/export shipments online
- Upload customs and shipping documents remotely
- Track cargo status in real-time
- Reduce the need for physical visits to Walvis Bay port or Windhoek customs offices

Especially beneficial for clients in rural and remote areas of Namibia.

## Key Features

- User registration and secure login (Customers & Admins)
- Create shipments with full sender/receiver and cargo details
- Unique tracking number generation (e.g., TON-2025-000001)
- Document upload and management per shipment
- Manual status updates by administrators
- Full shipment history timeline
- Public tracking page (no login required)
